3. Hata Kodu Sistemi

3.1 ErrorCode Tanımlama

public static class CustomerErrorCodes
{
    public static readonly ErrorCode CustomerNotFound = new(
        "CUST-001",
        "Müşteri bulunamadı",
        "Customer not found",
        404,
        ErrorCategory.NotFound);

    public static readonly ErrorCode EmailAlreadyExists = new(
        "CUST-002",
        "Bu email adresi zaten kayıtlı",
        "Email already exists",
        409,
        ErrorCategory.Conflict);
}

3.2 BusinessException Kullanımı

// Basit kullanım
throw new BusinessException(CustomerErrorCodes.CustomerNotFound);

// Orijinal hatayı koruyarak (KRİTİK!)
catch (Exception ex)
{
    // Önce logla, sonra BusinessException fırlat
    throw new BusinessException(
        errorCode,
        ex,  // Orijinal exception korunur
        new Dictionary<string, object>
        {
            ["ServiceName"] = "PaymentService",
            ["OriginalError"] = ex.Message
        });
}
Önemli: BusinessException fırlatırken orijinal exception'ı innerException olarak geçin!

4. Middleware Tabanlı Loglama

4.1 Eski vs Yeni Yaklaşım

Eski Yaklaşım (Kötü): 
public async Task<IActionResult> CreateCustomer(...)
{
    _logger.LogInformation("Started"); // Her metoda yazılıyor!
    try {
        ...
        _logger.LogInformation("Completed"); // Tekrar!
    } catch {
        _logger.LogError("Failed"); // Tekrar!
    }
}
Yeni Yaklaşım (İyi): 
public async Task<IActionResult> CreateCustomer(...)
{
    // Hiç log kodu yok - middleware otomatik hallediyor!
    var result = await _service.CreateAsync(request);
    return Ok(result);
}

4.2 Middleware Kullanımı

// Program.cs - Tek satır!
app.UseEnterpriseLogging();

// Bu middleware şunları otomatik loglar:
// - Request/Response (body dahil)
// - Action başlangıç/bitiş
// - Exception'lar
// - Performance metrikleri

5. Proxy Katmanı (WCF & HTTP)

5.1 WCF Proxy (Dispatcher ile Loglama)

public class PaymentProxy : WcfProxyBase<IPaymentService>
{
    public async Task<PaymentResult> ProcessAsync(PaymentRequest request)
    {
        // Otomatik:
        // - SOAP Request/Response loglama
        // - Correlation ID propagation
        // - Exception handling ve BusinessException dönüşümü
        return await ExecuteAsync(
            channel => channel.ProcessPaymentAsync(request),
            "ProcessPayment",
            PaymentErrorCodes.PaymentFailed);
    }
}
